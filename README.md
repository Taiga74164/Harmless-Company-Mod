# Non-Lethal-Company-Mod


### Requirements:
- [BepInEx](https://github.com/BepInEx/BepInEx)

### Usage:
1. Put DLL inside `\BepInEx\plugins`

### Features:
- Movement Speed
- No Clip
- No Fall Damage
- Unlimited Sprint
- No Weight
- God Mode
- Grab Distance
- Credits
- Teleport to scraps / Change scrap value
